# monitoring_deploy

## node_exporter客户端部署

  客户端的部署，通过shell脚本部署。

采集指标
##### 开启的采集指标
https://github.com/prometheus/node_exporter#enabled-by-default

 

##### 默认关闭可开启的采集指标
https://github.com/prometheus/node_exporter#disabled-by-default 


##### shell部署

获取安装脚本，本项目路径 node_exporter_install下。

分发到主机，执行脚本。

```sh
$ bash node_exporter_install.sh
```
